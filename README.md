# 🍜 Buldak Noodle House Website

A modern, fully functional website for ordering Korean Buldak noodles with a menu, shopping cart, and order management system.

## Features

✨ **Menu Display**
- Browse available noodle options
- View prices and descriptions
- Best seller highlights
- Add-on options (Korean Spam, etc.)

🛒 **Shopping Cart**
- Add items to cart
- Remove items
- Real-time cart total calculation
- Cart item counter

📦 **Order Management**
- Customer information form
- Delivery date selection
- Address input
- Order confirmation

🎨 **Design**
- Responsive layout (mobile, tablet, desktop)
- Modern UI with smooth animations
- Clean and intuitive navigation

## How to Use

1. **View Menu**: Check out the available noodle options on the menu page
2. **Add to Cart**: Click "Add to Cart" on any item you want to order
3. **Review Cart**: Navigate to the Cart tab to see your selections
4. **Order**: Click "Proceed to Checkout" and fill in your delivery details
5. **Confirm**: Submit your order and receive confirmation

## Installation

### Local Setup
1. Clone the repository
   ```bash
   git clone https://github.com/andreijanssen0911-coder/noodle-website.git
   cd noodle-website
   ```

2. Open in browser
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     python -m http.server 8000
     # Visit http://localhost:8000
     ```

### GitHub Pages
1. Go to repository Settings
2. Navigate to Pages
3. Set source to `main` branch
4. Your site will be live at: `https://andreijanssen0911-coder.github.io/noodle-website/`
